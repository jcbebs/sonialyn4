HCI FINAL EXAM PRACTICE QUIZ

Hard Multiple Choice Reviewer • Mobile Friendly

1. Which lifecycle activity primarily identifies separately implementable modules?
A. Architectural Design
B. Detailed Design
C. Requirements Specification
D. Validation
Detailed Design refines architectural components into modules for implementation.
2. Validation in software engineering refers to:
A. Ensuring code compiles successfully
B. Designing the product right
C. Designing the right product
D. Reducing development cost
Validation checks whether the correct product is being built for users.
3. Which prototyping approach evolves directly into the final system?
A. Throw-away
B. Wizard of Oz
C. Evolutionary
D. Incremental simulation
Evolutionary prototyping continuously improves until it becomes the final product.
4. Which ISO/IEC 25010 criterion measures ability to operate on different platforms?
A. Reliability
B. Portability
C. Maintainability
D. Compatibility
Portability refers to running software on different environments or platforms.
5. Which design rationale method is process-oriented?
A. QOC
B. IBIS
C. MVC
D. PAC
IBIS preserves order of deliberation and decision-making.
1. Which usability principle focuses on extending interaction knowledge to new situations?
A. Familiarity
B. Generalizability
C. Predictability
D. Synthesizability
Generalizability allows users to apply previous knowledge to new contexts.
2. Which flexibility principle supports simultaneous task execution?
A. Dialogue initiative
B. Substitutivity
C. Multithreading
D. Observability
Multithreading supports concurrent or interleaving tasks.
3. ISO 9241 defines usability using which three concepts?
A. Speed, utility, portability
B. Learnability, flexibility, robustness
C. Effectiveness, efficiency, satisfaction
D. Visibility, consistency, recoverability
ISO 9241 defines usability as effectiveness, efficiency, and satisfaction.
4. Which Norman principle emphasizes correct relationships between controls and effects?
A. Get the mappings right
B. Design for error
C. Simplify the structure of tasks
D. Standardize
Mappings determine how controls correspond to outcomes.
5. Patterns in HCI are primarily intended to:
A. Replace usability testing
B. Reuse successful design knowledge
C. Eliminate user feedback
D. Reduce programming complexity only
Patterns capture reusable successful solutions to recurring problems.
1. Which architecture provides maximum portability in windowing systems?
A. OS kernel management
B. Shared synchronization model
C. Separate application management
D. Centralized process control
Management as a separate application maximizes portability.
2. Which programming approach commonly uses callback handlers?
A. Event-loop programming
B. Notification-based programming
C. Sequential programming
D. Batch programming
Notification systems rely on callback handlers for events.
3. Swing is primarily based on which architecture?
A. PAC
B. Seeheim
C. MVC
D. IBIS
Swing toolkit uses the MVC architecture.
4. Which type of feedback includes menu highlighting?
A. Semantic
B. Syntactic
C. Lexical
D. Abstract
Syntactic feedback relates to interface structure and selections.
5. In MVC, which component manages screen rendering?
A. Controller
B. View
C. Model
D. Presenter
The View handles visual representation of data.
